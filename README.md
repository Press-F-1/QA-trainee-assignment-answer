# Task 2
## Тестирование API микросервиса Avito (QA Internship 2025) с 4 ручками
- **Создать объявления**
- **Получить объявления по его идентификатору**

- **Получить статистику по айтем id**

## Технологии

- Python 3.13.5
- Pytest 7.4.0
- Requests 2.31.0
- Стандартные библиотеки: re, random
- Virtual environment `venv`

## Содержание

- **`TESTCASES.md`** - составленные тест-кейсы в количестве 18-ти сценариев с обширным покрытием.
- **`tests.py`** - набор автотестов на Python + pytest.

## Зависимости

- **`requirements.txt`** - содержит необходимые модули.

### Установка зависимостей
**Пишем при переходе в папку репозитория в консоль**

1. `cd Task_2` - переходим в нужную папку.
2. `python -m venv my_venv` - создаём виртуальное окружение.
3. `source my_venv/bin/activate` - активируем виртуальное окружение.
4. `pip install -r requirements.txt` устанавливаем модули.

## Тестирование

**Проект покрыт юнит-тестами `pytest`**

- `pytest tests.py`
- `pytest -v tests.py` - для более подробного вывода.
